# How To Run

- First install the needed packages

```sh
pip install -r requirements.txt
```

- Add the needed data to test the model in the `data` folder next to the `src` folder

```sh
├───data
├───src
│   ├───main.py
│   └───predict.py
...
```

- Navigate to the src directory

```sh
cd src
```

- run the `main.py` file

```python
python main.py
```
